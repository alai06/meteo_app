# Angular Meteo App

Ce dépôt contient une petite application Angular pour consulter des prévisions météo.

## Contenu

- Code source : `src/`
- Service météo : `src/app/services/weather.service.ts`

## Prérequis

- Node.js (version compatible avec Angular 19)
- npm
- (optionnel) CLI GitHub `gh` si vous souhaitez créer un repo depuis le terminal

## Installation

Depuis la racine du projet, exécutez :

```bash
npm install
```

## Lancer l'application en mode développement

```bash
ng serve --open
```
